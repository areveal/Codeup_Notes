###Ternary Operator
This is crazy cool but crazy difficult to use and read!!

This is a shortcut for if else statements
```
[condition] ? [value if true] : [value is false];
```
example
```
$age = 22;
echo ($age >= 18) ? "yes!" : "no, try disney!";
```
will echo
```
yes!
```
